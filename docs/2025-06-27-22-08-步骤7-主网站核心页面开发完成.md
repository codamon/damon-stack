## 📱 响应式设计

### 断点策略
- **Mobile**: base - 单列布局
- **Tablet**: sm - 2列布局 
- **Desktop**: lg - 3-4列布局
- **Large**: xl - 最大宽度容器

### 组件适配
```typescript
// 网格系统示例
<SimpleGrid cols={{ base: 1, sm: 2, lg: 4 }} spacing="xl">
  {/* 响应式卡片 */}

<Container size="lg"> // 最大960px
```
